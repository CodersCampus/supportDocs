## keyboard shortcuts 
You hopefully already know:

Windows:
 cntrl + c (copy), cntrl + v (paste), cntrl + z (undo)

Mac:
cmd + c, cmd + v, cmd + z

And now make constant use of:

Eclipse:

- **Format Code:** `Cmd + Shift + F`

- **Save: `Cmd + S` (on mac)

- **Save: `Cntrl + S` (on windows)

IntelliJ:

- **Reformat Code:** `Cmd + Option + L`
