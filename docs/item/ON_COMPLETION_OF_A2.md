## keyboard shortcuts 
You hopefully already know cntrl + c (copy), cntrl + v (paste), cntrl + z (undo)
(on mac cmd + c, cmd + v, cmd + z)

Eclipse keyboard commands: 
- cntrl + space (code completion)
- cntrl + / (comment out/uncomment)

IntelliJ keyboard commands: 
- cntrl + space (code completion)
- cntrl + / (comment out/uncomment)
