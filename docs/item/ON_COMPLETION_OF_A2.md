## keyboard shortcuts 
You hopefully already know:

Windows:

Mac:
cmd + c, cmd + v, cmd + z

And by now you constantly utilize:
Eclipse keyboard commands: 
- cntrl + space (code completion)
- cntrl + / (comment out/uncomment)

IntelliJ keyboard commands: 
- cntrl + space (code completion)
